Description

In the critically acclaimed MMORPG Final Fantasy XIV (usually abbreviated in FFXIV) the class system is fundamentally different from the other games in the genre: instead of forcing the choice of only one role for each character the game gives the possibility to the player to change it on the fly, letting the player max the level on every playable class available at the moment. This type of system doesn't force the playerbase to replay the same content again and again just to enjoy the endgame with a different job.

In this site I will briefly explain the main features of each playable class, while providing some useful data and resources that might help the users to choose which job to add to their arsenal.
